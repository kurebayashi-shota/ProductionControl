# 生産管理

## 平準化(レベリング)

生産量や生産順序のバラつきをなおして、安定した生産にする考え方  
->ムダ・無理・ムラを減らすための中核概念

1. 生産平準化
    - 日毎の生産量を一定にすること
2. 品種平準化
    - 複数の製品のバランスよく混ぜて生産すること

## 調べた用語集
- 仕掛け:作業合図
- 仕掛り:作業途中
- 水位(仕掛り)を下げると岩(問題)が見える
- 背番号:工場、部品受入場所、メーカー名、背番号で部品を特定 [田原 N1 デンソー 背番号2]
- 設変:設計変更の略
- MRO:副資材
- MROS:なんかの連携?
- 連番:かんばんを一意で識別するための番号
- 差立て:作業指示
- 変化点:生産に変更があった時点のこと(主にM4に変更があるとき)
- だんご生産:まとめ生産=今の時代にあってない
- 納入かんばんと仕掛けかんばん
- 手番:1直=2,2直=48

# かんばん方式について
## かんばん方式とは
Pull生産方式(需要に基づいた生産方式)で[かんばん]を使用する  
- 特徴:かんばんとモノが同時に動く  
- 目的:QC改善・作業改善・在庫低減  
->運搬LTと滞留時間を短くできる
    - 実現するためには...
        1. 平準化  
            　:重みと量をバランスよく
        2. 内段取り時間の短縮  
        　  :生産品種の切り替えに発生する時間
        3. 生産速度の同期化(清流化)  
            　:機械配置と運搬時の導線を最適化する

※Push生産方式=どんどん作る
1. 需要の変化に対応しづらい
2. 計画範囲が広く管理工数が大きい
3. 在庫が過剰になりやすい
    - 前工程からどんどん送られてくる
    - 計画外の生産をしてしまう可能性も

## かんばんとは
引取・生産指示情報を伝えるツール
- 種類
    1. 生産指示かんばん
        - 生産かんばん:1箱単位で作る
        - 信号かんばん:ロット単位で作る
    2. 引取指示かんばん
        - 引取かんばん:前工程から引き取る
        - 外注かんばん:外注先から引き取る

### 生産かんばん
- 使用方法
    1. 後工程が引き取りにくるとかんばんが[はずれ]てポストへ
    2. ポストに入ったかんばんは定期的に回収されスケジュール板へ
    3. スケジュール板の順番でかんばんは工程と一緒に流して枚数分の生産をする
    4. 完成品に生産かんばんをつけて完成品ストアに格納

- 記載内容
    - 工程名  
        ->製造されている工程(例:加工セル#1)
    - ストア=店  
        ->完成品の格納場所
    - 背番号  
        ->自社内で製品を一意に識別する番号
    - 収容数  
        ->かんばん1枚に対する生産指示数

### 引取かんばん
- 使用方法
    1. 後工程で使う部品の引取かんばんが[はずれ]てポストへ
    2. 運搬担当が定期的にかんばんを回収して引取にいく
    3. かんばん枚数分だけストアから生産かんばんを[はずれ]てポストへ
    4. 生産かんばんの代わりに引取かんばんをつける

- 記載内容
    - 前後工程  
        ->どこからどこへ行くのかの情報
    - ストアと棚番号  
        ->前工程の格納ストア情報
    - 背番号  
        ->正式品番の代わりに簡略化した識別番号
    - 発行番号  
        ->再発行時などでかんばん識別の管理番号

### 外注引取かんばん
- 使用方法
    1. 納入時に発注棚から外注かんばんを持ち帰る
    2. 部品別・便別で仕分けられ棚へ入れられる
    3. 納入便に間に合うようにストアから集荷して外注かんばんをつける
    4. 納入時刻に間に合うように運搬する
    5. 受入場で荷下ろし->かんばんを持ち帰る

## スライド
納入日の順延->機器トラブルなどによりラインがとまった等
